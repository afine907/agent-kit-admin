# 文档规范

## 文档类型

| 类型 | 用途 | 内容 |
|---|---|---|
| README.md | 项目简介 | 目的、快速开始、安装、示例 |
| API 文档 | 接口说明 | OpenAPI 规范、请求/响应、错误码 |
| 架构文档 | 系统设计 | Mermaid 图、数据流、组件职责 |
| 开发文档 | 开发指南 | 环境搭建、规范、测试、部署 |

## 编写原则

- **清晰性**: 简单直接，避免歧义
- **完整性**: 覆盖功能、边界情况、故障排除
- **及时性**: 代码变更同步更新文档

## 中文文档规范

- 使用中文标点：，。！？；：""''（）
- 数字和英文使用半角字符
- 中英文之间添加空格
- 段落之间空一行，列表项之间不空行

## 新功能文档模板

```markdown
## 功能名称

### 概述
简短描述功能用途

### 使用方法
代码示例

### 配置选项
| 选项 | 类型 | 默认值 | 描述 |
|---|---|---|---|

### 注意事项
- 使用注意事项
- 已知限制
```
