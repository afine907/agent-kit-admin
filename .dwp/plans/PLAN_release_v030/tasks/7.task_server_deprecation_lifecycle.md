# Task 7: Server Deprecation Lifecycle

## Context

- `Version.deprecated` Boolean field already exists in `apps/server/app/models/version.py`
- No deprecation reason field exists yet
- No deprecate/undeprecate API endpoints exist

## Goal

Add deprecation with reason + un-deprecate endpoints.

## Instructions

### 1. Add deprecation_reason to Version model

File: `apps/server/app/models/version.py`

```python
deprecation_reason: Mapped[str | None] = mapped_column(Text, nullable=True)
```

### 2. Add migration

```bash
cd apps/server && alembic revision --autogenerate -m "add deprecation_reason to versions"
```

### 3. Add API routes

File: `apps/server/app/api/packages.py`

```
POST /packages/{scope}/{name}/deprecate
Body: {"reason": "Use @team/newpkg@^2.0 instead", "version": "1.0.0"}
→ marks specified version deprecated (or all versions if version omitted)

POST /packages/{scope}/{name}/undeprecate
Body: {"version": "1.0.0"}
→ un-deprecates specified version (or all versions if version omitted)
```

### 4. Update schemas

```python
class DeprecateRequest(BaseModel): reason: str; version: str | None = None
class UndeprecateRequest(BaseModel): version: str | None = None
```

PackageResponse: add `deprecation_reason: str | None` (returns first non-null reason if any version deprecated)

### 5. Write tests

File: `apps/server/tests/test_deprecation.py`

## Acceptance Criteria

- [ ] Deprecate with reason → Version.deprecated=True, deprecation_reason set
- [ ] Deprecate without version → all versions deprecated
- [ ] Undeprecate → deprecated=False, reason cleared
- [ ] PackageResponse shows `deprecated=true` and `deprecation_reason` when any version is deprecated
- [ ] Non-admin gets 403 on deprecate/undeprecate

## Validation

```bash
cd apps/server && pytest tests/test_deprecation.py -v
cd apps/server && ruff check app/
cd apps/server && mypy app/
```

## Outputs

- `apps/server/app/models/version.py` — deprecation_reason field
- `apps/server/app/api/packages.py` — deprecate/undeprecate routes
- `apps/server/app/schemas/package.py` — new schemas
- `apps/server/alembic/versions/XXX_add_deprecation_reason.py` — migration
- `apps/server/tests/test_deprecation.py` — tests
