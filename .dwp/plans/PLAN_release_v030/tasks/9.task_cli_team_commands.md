# Task 9: CLI Team Commands

## Context

- CLI in `apps/cli/src/commands/`
- Existing commands: install, publish, search, list, whoami, login, logout, info, init
- API client: `apps/cli/src/api/client.ts`
- Test runner: vitest (upgraded to 1.6.1)
- oclif-style command structure

## Goal

Add `akit team` subcommand group with: invite, join, members, role, settings.

## Instructions

### 1. Create team command file

File: `apps/cli/src/commands/team.ts`

Implement oclif-style command with subcommands:

```
akit team invite <team-name>
  → GET /teams?name=<team-name> → POST /teams/{team_id}/invites
  → Print: "Invite link: https://.../join?token=..."

akit team join <token>
  → POST /teams/join {"token": "..."}
  → Print: "Joined team successfully!"

akit team members <team-name>
  → GET /teams?name=<team-name> → GET /teams/{team_id}/members
  → Print table: USER | ROLE | JOINED

akit team role <user-id> --role <admin|member> [--team <team-name>]
  → PUT /teams/{team_id}/members/{user_id}/role {"role": "..."}
  → Print: "Role updated"

akit team settings <team-name> [--name X] [--description X] [--avatar X]
  → PUT /teams/{team_id}/settings
  → Print: "Settings updated"
```

### 2. Update command index

File: `apps/cli/src/commands/index.ts`

Add team command export.

### 3. Update CLI help

Ensure `akit --help` and `akit team --help` show the new subcommands.

### 4. Write tests

File: `apps/cli/tests/commands/team.test.ts`

Mock API client responses for each subcommand.

## Acceptance Criteria

- [ ] `akit team --help` lists all subcommands
- [ ] `akit team invite <team>` prints invite link
- [ ] `akit team join <token>` succeeds with valid token
- [ ] `akit team members <team>` prints member table
- [ ] `akit team role <user> --role admin` updates role
- [ ] `akit team settings <team> --name NewName` updates settings
- [ ] All commands handle errors gracefully (invalid token, not authorized, etc.)

## Validation

```bash
cd apps/cli && pnpm exec vitest run tests/commands/team.test.ts
cd apps/cli && pnpm exec oxlint src/commands/team.ts
cd apps/cli && pnpm exec tsc --noEmit
```
