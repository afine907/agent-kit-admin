# Task 1: Server Team Invite Flow

## Context

- Team/TeamMember models exist in `apps/server/app/models/team.py`
- TeamService in `apps/server/app/services/team.py` has `create_team`, `list_members`, etc.
- Teams API router in `apps/server/app/api/teams.py`
- Existing auth: Bearer token via `get_current_user`

## Goal

Implement invite token flow: generate invite → share link → accept invite → create TeamMember.

## Instructions

### 1. Add TeamInvite model

File: `apps/server/app/models/team.py`

```python
class TeamInvite(Base):
    __tablename__ = "team_invites"

    id: Mapped[uuid.UUID] = mapped_column(primary_key=True, default=uuid.uuid4)
    team_id: Mapped[uuid.UUID] = mapped_column(CompatUUID, ForeignKey("teams.id"), nullable=False, index=True)
    token: Mapped[str] = mapped_column(String(20), unique=True, nullable=False, index=True)
    created_by: Mapped[uuid.UUID] = mapped_column(CompatUUID, ForeignKey("users.id"), nullable=False)
    expires_at: Mapped[datetime] = mapped_column(DateTime, nullable=False)
    max_uses: Mapped[int] = mapped_column(Integer, default=1)
    use_count: Mapped[int] = mapped_column(Integer, default=0)
    created_at: Mapped[datetime] = mapped_column(DateTime, default=datetime.utcnow)
```

### 2. Add service methods in TeamService

File: `apps/server/app/services/team.py`

- `generate_invite(team_id, user_id, expires_hours=72, max_uses=1)` — generate 8-char alphanumeric token
- `list_invites(team_id)` — return active invites
- `revoke_invite(team_id, token, user_id)` — admin only, delete invite
- `accept_invite(token, user_id)` — validate not expired, use_count < max_uses → create TeamMember

### 3. Add API routes

File: `apps/server/app/api/teams.py`

- `POST /teams/{team_id}/invites` → create invite, return token + link
- `GET /teams/{team_id}/invites` → list active invites (admin+)
- `DELETE /teams/{team_id}/invites/{token}` → revoke invite (admin+)
- `POST /teams/join` → accept invite (body: `{"token": "..."}`)

### 4. Add schemas

File: `apps/server/app/schemas/team.py`

```python
class TeamInviteResponse(BaseModel): ...
class TeamInviteCreate(BaseModel): expires_hours=72, max_uses=1
class TeamJoinRequest(BaseModel): token: str
```

### 5. Add migration

```bash
cd apps/server && alembic revision --autogenerate -m "add team_invites table"
```

## Acceptance Criteria

- [ ] `POST /teams/{team_id}/invites` returns token + expiry
- [ ] `POST /teams/join` creates TeamMember and increments use_count
- [ ] Expired token → 400 error
- [ ] Exhausted token (max_uses reached) → 400 error
- [ ] Non-member cannot create invite (404 or 403)
- [ ] Admin can revoke invite
- [ ] New member can immediately publish/install under team
