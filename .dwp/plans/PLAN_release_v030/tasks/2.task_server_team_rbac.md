# Task 2: Server Team RBAC

## Context

- TeamMember.role: `owner` | `admin` | `member`
- TeamService already has: `_check_admin_permission`, `_check_owner_permission`, `_is_team_admin`, `_is_team_member`
- No role promotion/demotion/transfer endpoints exist yet

## Goal

Implement role management: promote/demote members, transfer ownership with acceptance.

## Instructions

### 1. Add PendingOwnershipTransfer model

File: `apps/server/app/models/team.py`

```python
class PendingOwnershipTransfer(Base):
    __tablename__ = "pending_ownership_transfers"
    team_id: Mapped[uuid.UUID] = mapped_column(CompatUUID, ForeignKey("teams.id"), primary_key=True)
    from_owner_id: Mapped[uuid.UUID] = mapped_column(CompatUUID, ForeignKey("users.id"), nullable=False)
    to_user_id: Mapped[uuid.UUID] = mapped_column(CompatUUID, ForeignKey("users.id"), nullable=False)
    created_at: Mapped[datetime] = mapped_column(DateTime, default=datetime.utcnow)
    expires_at: Mapped[datetime] = mapped_column(DateTime, nullable=False)  # 24h expiry
```

### 2. Add service methods in TeamService

File: `apps/server/app/services/team.py`

- `change_member_role(team_id, target_user_id, new_role, actor_user_id)` — validate permissions, update role
  - Owner can: member↔admin, admin→owner (via transfer), owner→member (only via transfer)
  - Admin can: member→admin only
- `initiate_ownership_transfer(team_id, from_user_id, to_user_id)` — create PendingOwnershipTransfer, send signal (for future email notification)
- `accept_ownership_transfer(team_id, user_id)` — verify pending transfer exists and not expired, update TeamMember roles: old owner→member, new owner→owner
- `cancel_ownership_transfer(team_id, user_id)` — cancel pending transfer

### 3. Add API routes

File: `apps/server/app/api/teams.py`

- `PUT /teams/{team_id}/members/{user_id}/role` — body: `{"role": "admin"|"member"}`
- `POST /teams/{team_id}/transfer-ownership` — body: `{"to_user_id": "..."}`
- `POST /teams/{team_id}/accept-ownership` — accept pending transfer
- `DELETE /teams/{team_id}/transfer-ownership` — cancel pending transfer

### 4. Add schemas

```python
class RoleChangeRequest(BaseModel): role: Literal["admin", "member"]
class TransferOwnershipRequest(BaseModel): to_user_id: UUID
class TransferOwnershipResponse(BaseModel): message: str; expires_at: datetime
```

### 5. Add migration

```bash
cd apps/server && alembic revision --autogenerate -m "add pending_ownership_transfers table"
```

## Acceptance Criteria

- [ ] Owner can promote member → admin
- [ ] Owner can demote admin → member
- [ ] Admin cannot promote anyone to owner
- [ ] Admin cannot demote other admins
- [ ] Owner initiates transfer → pending record created
- [ ] Target accepts transfer → roles swap correctly
- [ ] Expired transfer → 400 error on accept
- [ ] Self-demotion blocked (owner cannot demote themselves without transfer)
