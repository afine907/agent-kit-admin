# Task 14: Web Package Category Filter

## Context

- Home page: `apps/web/src/pages/Home.tsx` with package search/filter
- PackageCard component shows basic info
- URL params used for pagination

## Goal

Add category dropdown filter and tag cloud to home page.

## Instructions

### 1. Add category dropdown to Home.tsx

In the search/filter bar above package grid:

```tsx
<select onChange={(e) => setUrlParams({...params, category: e.target.value})}>
  <option value="">All Categories</option>
  <option value="mcp">MCP</option>
  <option value="skill">Skill</option>
  <option value="tool">Tool</option>
  <option value="integration">Integration</option>
</select>
```

Fetch categories from GET /packages/tags?field=category (or derive from package data).

### 2. Add tag cloud to sidebar

```tsx
<div className="tag-cloud">
  {allTags.map(tag => (
    <button
      key={tag}
      onClick={() => setUrlParams({...params, tag})}
      className={selectedTag === tag ? 'tag-selected' : 'tag'}
    >
      {tag} ({tagCount})
    </button>
  ))}
</div>
```

Fetch tags: GET /packages/tags → list of {tag, count}.

### 3. Selected tag/category chips

When filter is active, show removable chip:

```tsx
{selectedTag && (
  <div className="flex items-center gap-2">
    Tag: {selectedTag}
    <button onClick={() => removeTagFilter()}>×</button>
  </div>
)}
```

### 4. Category badge on PackageCard

In `PackageCard.tsx`, show category badge if present:

```tsx
{category && (
  <span className="text-xs bg-mcp/10 text-mcp px-2 py-0.5 rounded">
    {category}
  </span>
)}
```

### 5. URL params integration

Use existing URL param pattern:

```tsx
const [params, setParams] = useSearchParams()
// params.get('tag'), params.get('category')
// Update API call: GET /packages?tag=mcp&category=mcp
```

## Acceptance Criteria

- [ ] Category dropdown appears in filter bar and filters package list
- [ ] Tag cloud appears in sidebar with all tags + counts
- [ ] Clicking tag filters list to that tag
- [ ] Active tag shown as removable chip
- [ ] Category badge shown on PackageCard
- [ ] Filters combine (category + tag + search)
- [ ] Filtered URLs are shareable
