# Task 8: Fix web TeamPackagesTab buttons

## Context

The "Publish Package" button (line 108) and "Delete" trash icon button (line 232) in `TeamPackagesTab.tsx` have no `onClick` handlers. They render but do nothing.

## Read Before Starting

- `apps/web/src/components/TeamPackagesTab.tsx` — broken buttons
- `apps/web/src/lib/api.ts` — API methods available

## Goal

Make Publish and Delete buttons functional with proper user feedback.

## Instructions

1. **Publish button:** Add `onClick` handler that navigates to `/publish?scope=team:{teamId}` (or equivalent)
2. **Delete button:** Add `onClick` handler that:
   - Shows a confirmation dialog (use shadcn Dialog, not native `confirm()`)
   - Calls `api.deleteTeamPackage(teamId, packageId)` on confirmation

   - Shows success/error toast
