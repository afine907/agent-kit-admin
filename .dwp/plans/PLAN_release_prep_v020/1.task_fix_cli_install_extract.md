## Goal

Make `akit install` work end-to-end: download → extract → read manifest → check dependencies → write agent config.

## Instructions

1. Add `extractTarball(tarballPath: string, destDir: string): Promise<void>` to `apps/cli/src/utils/tarball.ts`
