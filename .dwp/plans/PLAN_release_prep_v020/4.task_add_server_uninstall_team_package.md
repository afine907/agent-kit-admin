# Task 4: Add server uninstall-team-package endpoint

## Context

There is a `POST /{team_id}/packages/{package_id}/install` endpoint but no corresponding uninstall. The `InstalledPackage` model has no soft-delete or removal mechanism.

## Read Before Starting

- `apps/server/app/api/teams.py` — install endpoint at line 407
- `apps/server/app/models/team.py` — InstalledPackage model
- `apps/server/app/services/teams.py` — team service

## Goal

Add `DELETE /api/v1/teams/{id}/packages/{pid}/install` to remove an installed team package record.

## Instructions

1. Add `uninstall_team_package(team_id, package_id, user)` to `apps/server/app/services/teams.py`:
   - Find the InstalledPackage record
   - Delete it (or soft-delete if the model supports it)
   - Verify user has permission (must be team member)
2. Add `DELETE /api/v1/teams/{id}/packages/{pid}/install` to `apps/server/app/api/teams.py`
3. Add a test for the uninstall endpoint
