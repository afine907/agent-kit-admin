# Task 6: CLI batch operations support

## 1. Context

CLI has no batch commands. Users need to deprecate or delete multiple packages from the terminal.

**Files to create:**
- `apps/cli/src/commands/batch.ts` — new command file

## 2. Goal

CLI commands for batch package operations.

## 3. Instructions

1. `akit batch delete @scope/pkg1 @scope/pkg2 ...` — batch delete
2. `akit batch deprecate @scope/pkg1 @scope/pkg2 ...` — batch deprecate
3. `--undeprecate` flag to un-deprecate
4. Confirmation prompt before execution (show package list)
5. `--yes` flag to skip confirmation
6. Output: per-package success/failure summary
7. Write tests

## 4. Acceptance Criteria

- Commands work with confirmation prompt
- `--yes` skips prompt
- Summary shows per-package results
- `--undeprecate` works
- Tests pass
