# Task 5: CLI webhook management commands

## 1. Context

CLI has no webhook commands. Users need to manage team webhooks from the terminal.

**Files to create:**
- `apps/cli/src/commands/webhook.ts` — new command file

## 2. Goal

CLI commands to list, create, and delete team webhooks.

## 3. Instructions

1. `akit webhook list --team @team` — list team webhooks in table format
2. `akit webhook add --team @team --url https://... --events publish,delete` — create
3. `akit webhook remove --team @team --id <id>` — delete
4. Register command in CLI entry point (`apps/cli/src/index.ts`)
5. Use existing API client patterns from other commands
6. Write tests in `apps/cli/tests/commands/webhook.test.ts`

## 4. Acceptance Criteria

- All three commands work with proper output
- Error handling for invalid team, permission denied
- Table output for list command
- Confirmation for remove command
- Tests pass
