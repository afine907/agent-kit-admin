# Task 1: Measure and establish test coverage baselines

## 1. Context

pytest-cov is in pyproject.toml but actual coverage is unknown. Need to establish baselines before improving.

**Files to modify:**
- `apps/server/pyproject.toml` — ensure coverage config
- `apps/cli/vitest.config.ts` — add coverage config
- `apps/web/vitest.config.ts` — add coverage config
- `docs/TESTING_GUIDE.md` — document baselines

## 2. Goal

Run coverage reports for all three components, document current state, set thresholds.

## 3. Instructions

1. Server: `cd apps/server && pytest --cov=app --cov-report=term-missing --cov-report=html`
2. CLI: `cd apps/cli && pnpm test -- --coverage`
3. Web: `cd apps/web && pnpm test -- --coverage`
4. Document results in `docs/TESTING_GUIDE.md` under "Coverage" section
5. Set coverage thresholds: server 80%, cli 70%, web 70%
6. Ensure `htmlcov/` and `coverage/` are in `.gitignore`

## 4. Acceptance Criteria

- Coverage reports generated for all three components
- Current percentages documented
- Thresholds configured
- No existing tests broken
- `.gitignore` updated

## 5. Validation

```bash
cd apps/server && pytest --cov=app --cov-fail-under=0
cd apps/cli && pnpm test
cd apps/web && pnpm test
```
