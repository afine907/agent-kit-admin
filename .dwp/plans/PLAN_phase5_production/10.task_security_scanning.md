# Task 10: Security scanning setup

## 1. Context

No automated security scanning in CI.

## 2. Goal

Set up dependabot, Trivy, pip-audit, pnpm audit in CI.

## 3. Instructions

1. Create `.github/dependabot.yml` — weekly updates for pip, npm, docker
2. Create `.github/workflows/security.yml`:
   - Trivy container scan
   - `pip-audit` for Python deps
   - `pnpm audit` for Node deps
3. Run audit locally to check current state

## 4. Acceptance Criteria

- `dependabot.yml` configured
- CI workflow runs Trivy, pip-audit, pnpm audit
- No critical vulnerabilities in current deps (or documented)

## 5. Validation

```bash
cat .github/dependabot.yml | grep -q "package-ecosystem"
cat .github/workflows/security.yml | grep -q "trivy"
```
