## 3. Instructions

> **Before starting:** Re-read the plan README section 1 (Goal) to ensure this task's work aligns with the overall objective.

### 测试用例

1. **正常离开团队** — 成员调用 leave，返回 204，成员从团队移除
2. **团队 owner 不能离开** — owner 调用 leave，返回 400，错误码 GUARD
3. **非成员不能离开** — 非团队成员调用 leave，返回 404
4. **未认证请求** — 无 token 调用 leave，返回 401
5. **团队不存在** — 调用不存在的团队 ID，返回 404
6. **离开后再次离开** — 已离开的成员再次调用，返回 404

### 实现方式

- 扩展 `apps/server/tests/test_team.py` 或新建 `test_leave_team.py`
- 使用 conftest.py 中的 fixtures（db_session, test_user, test_team 等）
- 使用 FastAPI TestClient 或 httpx AsyncClient
