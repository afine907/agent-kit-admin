### 测试用例

1. **正常卸载** — 团队成员卸载已安装的包，返回 204
2. **未安装的包** — 卸载未安装的包，返回 404
3. **非团队成员不能卸载** — 非成员调用，返回 403
4. **未认证请求** — 无 token，返回 401
5. **团队不存在** — 无效团队 ID，返回 404
6. **包不存在** — 无效包 ID，返回 404

### 实现方式

- 扩展 `apps/server/tests/test_team_packages.py` 或新建 `test_uninstall_team_package.py`
- 使用 conftest.py 中的 fixtures
