# Task 7: Web OAuth callback 测试

## 1. Context

- 影响范围：`apps/web/src/pages/AuthCallback.tsx`, `apps/web/src/App.tsx`
- 测试框架：Vitest + @testing-library/react
- v0.2.0 新增 OAuth callback 页面，处理 `/auth/callback?token=...` 路由

## 2. Goal

为 AuthCallback 组件编写全面的单元测试，覆盖正常流程和错误处理。

## 3. Instructions

> **Before starting:** Re-read the plan README section 1 (Goal) to ensure this task's work aligns with the overall objective.

### 测试用例

1. **正常 token 提取** — URL 包含 token 参数，验证 token 存入 localStorage
2. **登录后重定向** — token 存储后验证重定向到首页
3. **缺少 token** — URL 无 token 参数，验证错误处理（显示错误或重定向到登录页）
4. **空 token** — token 参数为空字符串，验证错误处理
5. **token 格式验证** — 验证 token 被正确存储（非 undefined/null）

### Mock 策略

- Mock `useNavigate` 验证重定向
- Mock `localStorage` 验证 token 存储
- Mock `useSearchParams` 控制 URL 参数
