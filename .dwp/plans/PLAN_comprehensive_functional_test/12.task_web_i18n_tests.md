# Task 12: Web i18n 覆盖率测试

## 1. Context

- 影响范围：`apps/web/public/locales/` 目录下的 locale 文件
- 测试框架：Vitest
- v0.2.0 修复了所有硬编码字符串，统一使用 i18n `t()` 调用

## 2. Goal

编写测试验证 i18n 覆盖率，确保中英文 locale 文件 key 一致。

## 3. Instructions

> **Before starting:** Re-read the plan README section 1 (Goal) to ensure this task's work aligns with the overall objective.

### 测试用例

1. **中英文 key 一致性** — 读取 zh/en locale 文件，验证所有 key 完全匹配
2. **无硬编码字符串** — 扫描组件源码，检测可能的硬编码中文/英文字符串（排除注释和 import）
3. **locale 文件完整性** — 验证所有 locale 文件是有效 JSON
4. **新增 key 覆盖** — 验证 v0.2.0 新增的功能都有对应的 i18n key

### 实现方式

- 创建 `apps/web/src/__tests__/i18n.test.ts`
- 使用 fs 读取 locale 文件进行比较
- 使用正则表达式扫描源码

## 4. Acceptance Criteria

- [ ] 覆盖 ≥4 个测试用例
- [ ] 所有测试通过
- [ ] 中英文 key 100% 一致
